# Administration

## What is it?

The pages in this section describe how to configure the Tuxedo ART Agent through the **Agent.config** file in the installation root directory and how to start and stop the agent using the **SMA_tux_agent** script. The configuration controls how the agent communicates with OpCon, how it captures job output, how it handles MSGIN event submission, and how it connects to the Tuxedo environment.

## In this section

- [Agent.config file configuration](./configuration-file.md) — Settings for general operation, TCP/IP communication, process creation, debug options, JORS, and the application connector.
- [Agent commands](./agent-commands.md) — Use the **SMA_tux_agent** script to start and stop the agent.
