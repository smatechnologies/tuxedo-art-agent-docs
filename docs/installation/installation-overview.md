# Installation

## What is it?

The pages in this section describe how to install and upgrade the Tuxedo ART Agent on a UNIX or Linux system, and how to define the corresponding machine in OpCon. The agent installation consists of copying the supplied tar file to the server, extracting it, and editing the installation scripts and **Agent.config** file to match the target Tuxedo environment.

## In this section

- [Installation introduction](./introduction.md) — How the Tuxedo ART Agent runs jobs and returns information to OpCon.
- [New installation](./new-installation.md) — Install the Tuxedo ART Agent on a UNIX or Linux system and create the machine record in OpCon.
- [Upgrade installation](./upgrade-installation.md) — Upgrade an existing Tuxedo ART Agent installation in place.
