# MSGIN messages

## What is it?

This page lists the messages the Tuxedo ART Agent writes when it processes events received through the **MSGIN** capability.

|Message|Description|
|---|---|
|Received event {0} via MSGIN | The following event has been received from MSGIN. |
|Received event {0} is not a valid format | The event is not a valid OpCon event. |
