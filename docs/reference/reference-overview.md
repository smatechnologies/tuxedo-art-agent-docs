# Reference

## What is it?

The pages in this section list the codes and messages the Tuxedo ART Agent reports during operation. Use them to interpret values returned in OpCon job output, to verify that the agent has started correctly, and to confirm that MSGIN events and job processing messages are being recorded as expected.

## In this section

- [Job return codes](./job-return-codes.md) — Return code values produced by `artjesadmin` and by Tuxedo ART job completion, including system and user abend ranges.
- [Startup messages](./startup-messages.md) — Messages the agent records when it starts, confirming the JES root directory, agent version, MSGIN file watcher, and substitution file directory.
- [MSGIN messages](./msgin-messages.md) — Messages recorded when the MSGIN file watcher receives event files.
- [Processing messages](./processing-messages.md) — Messages the agent records during job submission, status checks, and completion.
